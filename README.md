# it-mermaid-viewer

Interactive Mermaid Viewer.

## Features

- Interactive controller for [Sequence diagrams](https://mermaid.js.org/syntax/sequenceDiagram.html)
  - Enable `autonumber` option by default
- Zoom in/out
- Support embed
  - `?text=<url-encoded-mermaid-text>` query parameter
